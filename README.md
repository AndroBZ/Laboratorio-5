# Laboratorio-5
El ultimo lab!!!
Abra una terminal en la raíz del proyecto y ejecute la siguiente secuencia de comandos:

1. Limpiar ejecutables y archivos objeto previos
make clean

2. Compilar el proyecto de forma automatizada
make

3. Ejecutar el programa de pruebas unificado para efectuar las pruebas de todas las estructuras de datos
./programa_pruebas
